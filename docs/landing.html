核心亮点

为隐私和体验而生

不是简单的语音转文字。EchoFlow 接入 Fcitx5 输入法框架，像你用拼音打字一样精准上屏。

离线识别，隐私安全

录音和识别全部在本机完成，不经过任何云端服务。你的声音永远不会离开你的设备。

右 Ctrl 即按即说

焦点在输入框时，轻点右 Ctrl 开始录音，再点一下结束。无需离开键盘，无需切换窗口。

Fcitx5 原生上屏

通过 commitString 提交到当前输入框，不走剪贴板、不模拟按键，永远不会打错窗口。

模型常驻，说出即识别

模型在服务启动时加载并常驻内存，无冷启动等待。录音中后台分段转写，停止即得结果。

录音中实时预览

语音胶囊提示里显示临时转写结果，让你确认系统正在「听」你说话，停止时文本已就绪。

应用内一键下载模型

设置界面点击下载，支持 hf-mirror 和 HuggingFace 官方两种源，进度条清晰可见。

工作流程

从开口到上屏，一气呵成

四个步骤，全程无需离开键盘。所有组件通过 Unix domain socket 在本机通信，无需网络。

1

按下右 Ctrl

焦点在输入框时轻点右 Ctrl，语音胶囊立即出现。

2

开始说话

Qwen ASR 在后台实时分段转写，胶囊显示临时结果。

3

再按结束

再次轻点右 Ctrl，后台完成最终转写并合并整段文本。

4

文字上屏

通过 Fcitx5 原生通道提交到当前输入框，精准无误。

系统架构

三个组件，各司其职

守护进程、Fcitx5 插件、桌面 UI 通过本地 socket 协作，构成完整的语音输入闭环。

快速开始

三步开始用

  1. 克隆并构建

    需要 deepin 25.1+、Fcitx5、PipeWire。递归拉取子模块后用 CMake 构建。

  2. 本地安装

    运行安装脚本，自动启用两个用户 systemd 服务，并重载 Fcitx 使插件立即生效。

  3. 下载模型并使用

    托盘 → 设置 → 模型，选择 Qwen3-ASR，下载后轻点右 Ctrl 即可开始说话。
